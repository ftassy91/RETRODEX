Execute the approved lot only.

Mode: BUILD.

## Before writing any code

1. Restate the approved scope from the plan
2. Confirm the model is appropriate
3. List the files you will touch

## During execution

- Touch only listed files
- Out of scope discoveries: note them, do not fix them
- Keep changes minimal

## Return format

LOT:              [name]
SCOPE RESTATED:   [from plan]
FILES CHANGED:    [file: what + why, 1 line each]
CHECKS TO RUN:    [list]
REMAINING RISKS:  [list]
SCOPE DRIFT:      [none or describe]
MODEL FEEDBACK:   [appropriate? switch next?]
