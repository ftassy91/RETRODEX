Top Screen manual artwork dropzone

Purpose
- Store local exception artwork files for games that cannot be resolved cleanly through the automated Wikipedia pipeline.

Rules
- Prefer official artwork, key art, promo illustration, or editorial game art.
- Avoid full box scans and box-frame visuals when a cleaner illustration exists.
- Keep filenames aligned with the game id when possible.

Example
- `assets/top-screen/power-golf-turbografx-16.png`

After adding a file
- Register it in `data/top_screen_artwork.js` under `window.TOP_SCREEN_ARTWORK_DATA.manual`.
