# Notion Exports

This folder stores local JSON snapshots exported by the optional Notion connector.

Examples:

- `database_<id>.json`
- `data_source_<id>.json`
- `query_<id>.json`
- `page_<id>.json`

These files are intended as:

- local backups
- sync intermediates
- future import sources for Retrodex or RetroMarket
