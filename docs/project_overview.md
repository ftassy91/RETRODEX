# RetroDex - Project Overview
RetroDex est un hub centralise pour collectionneurs de jeux video retro.

## Donnees
- 507 jeux, 16 consoles
- Prix Loose / CIB / Mint pour chaque jeu
- 120 ventes verifiees RetroMarket

## Modules
- Backend port 3000 : API REST, SQLite
- Frontend port 8080 : prototype 3DS, RetroMarket CRT
