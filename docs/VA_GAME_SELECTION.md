# Vision A — Selection des 40 jeux pour anecdotes BAZ

## Criteres de selection
1. Popularite culturelle (les jeux que tout collectionneur connait)
2. Consoles du persona (NES, Game Boy, SNES, Genesis, N64, PS1, Saturn, Dreamcast)
3. Richesse anecdotique (histoire de dev, easter eggs, faits culturels)
4. Equilibre : pas que du SNES, pas que du RPG
